# 🏥 Sistema Omnichannel para Clínica Médica

Este projeto apresenta o desenvolvimento de um protótipo funcional de uma plataforma **Omnichannel** voltada para clínicas médicas, integrando múltiplos canais de comunicação com um banco de dados centralizado.

O trabalho foi desenvolvido para a matéria de **Low-Code** na **UNIC** e foca na viabilidade da centralização de dados e na agilidade da prototipagem.

## 🛠️ Tecnologias e Ferramentas
* **Linguagens:** PHP e SQL.
* **Banco de Dados:** MySQL Workbench.
* **Ambiente Local:** XAMPP (Apache e MySQL).
* **Editor:** VS Code.
* **Abordagem:** Híbrida (Tradicional + Low-Code assistido por IA).

## 🏗️ Arquitetura do Sistema
O sistema utiliza uma estrutura de dados robusta para garantir a fluidez do atendimento:
* **Tabela de Pacientes:** Registra o canal preferido de contato.
* **Controle de Acesso:** Perfis distintos para Atendentes, Gerentes e Pacientes.
* **Histórico Unificado:** Tabela de conversas que permite transferir atendimentos entre canais sem perda de contexto.

## 📊 Análise de Metodologia
O estudo reflete sobre a complexidade da integração entre plataformas de baixo código (como Bubble e Base44) e bancos de dados SQL externos, concluindo que o conhecimento em lógica de backend continua essencial para a escalabilidade.
